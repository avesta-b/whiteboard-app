Version 1.0.0 Release Notes
Compatibility: macOS, Linux, Windows 11, Windows 10

- Added ability to create a new account and sign in
- Added the ability to create a new whiteboard
- Can draw rectangles, circles, text boxes, and can perform free hand drawing
- Can connect to a room with other users
- Implemented window size persistence
- Added a menu bar, can now minimize and toggle fullscreen from the menu bar
